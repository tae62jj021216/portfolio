# 김태윤 Portfolio

항공소프트웨어, UAV 보안, 항공보안 및 방위산업 분야 취업 준비를 위한 정적 포트폴리오 웹사이트입니다.

## 구성

- `index.html`: 페이지 구조
- `styles.css`: 반응형 스타일
- `script.js`: 포트폴리오 데이터와 렌더링 로직
- `assets/hero-portfolio.png`: 메인 이미지

## 수정 방법

포트폴리오 내용은 `script.js` 상단의 `portfolio` 객체에서 수정합니다.

수정하기 좋은 항목:

- 기본 소개
- 프로젝트
- 기술과 강점
- 자격증 및 취득 계획
- 준비 일정
- 연락처

## 배포

정적 파일만 사용하므로 GitHub Pages로 배포할 수 있습니다.
