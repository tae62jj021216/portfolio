# AI Portfolio Website

AI를 활용해 정리한 개인 포트폴리오용 정적 웹사이트입니다. 별도 빌드 과정 없이 `index.html`을 브라우저에서 열면 바로 확인할 수 있습니다.

## 수정 방법

1. `script.js` 상단의 `portfolio` 객체에서 이름, 소개, 프로젝트, 기술, 연락처를 바꿉니다.
2. 대표 이미지를 바꾸고 싶으면 `assets/hero-portfolio.png`를 같은 파일명으로 교체합니다.
3. `index.html`을 브라우저에서 열어 화면을 확인합니다.

## 배포

정적 파일만 사용하므로 GitHub Pages, Netlify, Vercel 등에 그대로 올릴 수 있습니다.

필수 파일:

- `index.html`
- `styles.css`
- `script.js`
- `assets/hero-portfolio.png`
